# **App Name**: Nexus Hub

## Core Features:

- Dashboard Overview: Dashboard providing a clear overview of community stats such as total members, families, and expected contributions.
- Member Management: Members directory with filtering and search functionality to easily manage community members.
- Family Management: Family directory allowing the admin to view a summary of each family in the community and add new members to the family.
- Settings Management: Admin can configure membership tiers and contribution settings.
- Custom Contribution Templates: Enable admin to manually apply predefined discounts on specific members. For example student discount and sponsoring discounts.

## Style Guidelines:

- Primary color: Indigo (#4F46E5) to convey trust and stability.
- Background color: Very light gray (#F9FAFB), near-white, for a clean, modern aesthetic.
- Accent color: Teal (#319795) for interactive elements and highlights, providing a fresh and engaging feel.
- Body and headline font: 'Inter', a sans-serif font, will be used throughout the app for its modern, neutral, and highly readable design.
- Use consistent and clear icons from Lucide React to represent different functions and data categories, ensuring intuitive navigation and visual appeal.
- Implement a responsive and well-organized layout to ensure optimal usability across various screen sizes, with a focus on clear data presentation and easy access to key functions.
- Use subtle animations and transitions to enhance user experience, such as smooth modal appearances and hover effects on interactive elements.
